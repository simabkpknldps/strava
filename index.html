Bookmarklet Ekstrak Leaderboard

Simpan tombol di bawah sebagai bookmark. Setelah itu, buka halaman leaderboard klub Strava manapun (yang sudah kamu login) dan klik bookmark-nya — tabel leaderboard langsung diekstrak dan dibuka rapi di tab baru, siap dicetak atau disimpan sebagai PDF. Tidak perlu save file atau upload manual lagi.

1 Simpan bookmarklet-nya

Tampilkan bar bookmark browser kamu (biasanya Ctrl/Cmd + Shift + B), lalu seret tombol oranye di bawah ini ke bar bookmark.

📥 Ekstrak Leaderboard Strava ← seret ke bookmark bar
Drag-and-drop tidak berhasil? Buat manual
  1. Klik kanan pada bookmark bar browsermu → pilih "Add page" / "Tambah bookmark".
  2. Isi Nama: Ekstrak Leaderboard Strava
  3. Isi URL dengan kode di bawah ini (copy semuanya, termasuk kata javascript: di depan):

2 Buka halaman leaderboard klub

Login ke Strava seperti biasa, lalu buka halaman leaderboard klub yang ingin diekstrak, misalnya:

https://www.strava.com/clubs/<id-klub>/leaderboard

3 Klik bookmark-nya

Klik bookmark "Ekstrak Leaderboard Strava" yang sudah kamu simpan. Tab baru akan terbuka otomatis berisi tabel leaderboard yang rapi, lengkap dengan tombol 🖨️ Cetak / Simpan PDF.

Jika tab baru tidak muncul, kemungkinan browser memblokir popup untuk strava.com. Izinkan popup untuk situs tersebut lalu klik bookmark-nya sekali lagi.

Catatan & batasan penting

Kode ini berjalan sepenuhnya di browser kamu sendiri, di dalam halaman Strava yang sedang terbuka — tidak ada data yang dikirim ke server manapun. Karena itu:

  1. Kamu harus dalam keadaan sudah login ke Strava di tab tersebut.
  2. Bookmarklet ini dibuat berdasarkan struktur halaman leaderboard Strava saat ini; jika Strava mengubah tampilan HTML-nya di kemudian hari, ekstraksi mungkin perlu disesuaikan lagi.
  3. Ini murni alat bantu ekspor tampilan untuk pemakaian pribadi/klub — bukan pengganti API resmi Strava.
